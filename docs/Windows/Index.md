### U盘自定义ico

图片转换  [转换 (onlineconvertfree.com)](https://onlineconvertfree.com/zh/complete/svg-ico/)

根目录新建 `Autorun.inf` 文件 写入如下文件，并将ico文件重命名为`disk.ico`

```ini
[autorun]
icon=disk.ico
```

隐藏图片和配置文件 `attrib` 命令

```powershell
# R 只读文件属性。
# A 存档文件属性。
# S 系统文件属性。
# H 隐藏文件属性。

attrib +s +h +r 
```
